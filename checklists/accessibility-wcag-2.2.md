# Accessibility – WCAG 2.2 AA Essentials

- Keyboard accessible: all interactive elements focusable/operable
- Visible focus: clear, non-obscured focus indicators
- Logical focus order: matches visual and reading order
- ARIA roles/labels: present and appropriate; name/role/value exposed
- Text contrast: normal text ≥ 4.5:1, large text ≥ 3:1
- Non-text contrast: UI components/graphics meet contrast guidance
- Alt text: meaningful for images; decorative marked accordingly
- Forms: labels, errors, instructions, programmatic association
- Motion/animation: reduce/disable on prefers-reduced-motion
