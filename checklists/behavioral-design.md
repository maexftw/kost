# Behavioral Design – Fogg MAP, Cialdini (ethical)

## Fogg Behavior Model (B = MAP)
- Motivation: adequate for the task
- Ability: reduce friction (time, money, effort, routine)
- Prompt: timely, contextual triggers

Checklist
- Remove unnecessary steps/fields
- Provide defaults and examples
- Time prompts to user intent

## Cialdini Principles (use responsibly)
- Social proof (reviews, usage numbers)
- Authority (trusted endorsements)
- Reciprocity (value before request)
- Consistency/Commitment (progress trackers)
- Scarcity (only if factual)

No dark patterns. Be transparent and opt‑in.
